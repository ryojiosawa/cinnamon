OrgUnderTest

global with sharing class OrgUnderTest
DescriptionOrgUnderTest is a class that provides a set of utility methods to execute operations against an org under test. All operations from this class are executed via Salesforce Tooling or REST API

This class must be used when an org that tests are running against is differnt from the one where Cinnamon is installed.
