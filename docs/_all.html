VisualforceByIdLocator

global with sharing class VisualforceByIdLocator extends ElementLocator
DescriptionVisualforceByIdLocator is a subclass of ElementLocator and is used to locate an HTML element generated by a Visualforce tag. It is normally used with getElement method in PageObject especailly when you want to locate an element not supported by VisualforceLocator.

The following code snippet shows how to retrieve a form WebElmenet generated by <apex:form> tag with 'storyWorkForm' set to Id attribute. Because VisualforceLocator currently does not support <apex:form> tag, 'form' HTML tag is passed in as the first argument, instead. 

 
 cinnamon.WebElement storyWorkForm = getElement(new cinnamon.VisualforceByIdLocator('form', 'storyWorkForm'));

VisualforceLocator

global with sharing class VisualforceLocator extends ElementLocator
DescriptionVisualforceLocator is used to locate an HTML element generated by a Visualforce tag. It is normally used as an arguement to getElement method in PageObject class to find and return a WebElement object that represents a certain HTML element

The following Visualforce tags are currently supported by VisualforceLocator:

  • apex:selectList
  • apex:outputField
  • apex:pageBlockTable
  • apex:commandButton
  • apex:pageMessages
  • apex:inputField
For other Visualforce tags not listed above, VisualforceByIdLocator must be used to locate an element, instead.
SeeWebElement, VisualforceByIdLocator
